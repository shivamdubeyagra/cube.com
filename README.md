# Getting Started
## Installation
```
git clone https://github.com/shivamdubeyagra/cube.com.git
```

## Package Installation
```
npm i
```
## Run Project
```
npm run dev
```
# You should Know about these Tech Stack
- React
- TypScript
- HTML
- tailwindcss
- react-router-dom
